* 项目开始
  * 执行spider_main.py即可

* 对应文件说明
  * html_downloader.py  网页下载模块，使用urllib2下载
  * html_outputer.py  抓取到的数据保存在html文件内
  * html_parser.py  网页分析模块，调用BeautifulSoup
  * output.html 保存抓取到的网页数据
  * spider_main.py  主运行目录
  * url_manager.py  url地址管理模块
